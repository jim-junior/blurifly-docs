## Image Component

The Image component is used to create responsive images the are also help in page loading since the are optimized and implement lazy loading and are asyncromous. This means all necessary page data can be archived easily.
It is actually the component that can be customized more so you can change the component settings.

## Usage

To use the image component you haveto import it r=from the `reactjs-media` module as `Image`

**Example**

```jsx live
import React from "react";
import { Image } from "reactjs-media";

const MyImage = () => {
	return (
		<>
			<Image
				src="/image.jpg"
				//you can pass in other props
			/>
		</>
	);
};
```
